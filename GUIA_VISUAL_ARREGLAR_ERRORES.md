```powershell
# Elimina todas las carpetas bin
Get-ChildItem -Directory -Name "bin" -Recurse | ForEach-Object {

}

# Elimina todas las carpetas obj
Get-ChildItem -Directory -Name "obj" -Recurse | ForEach-Object {
```

## ?? CASOS COMUNES

### Caso 1: "Entidad_BE.dll no encontrado"

```
CAUSA: Entidad_BE no se compiló correctamente
SOLUCIÓN:
  1. Haz click derecho en Entidad_BE en el Explorador de Soluciones
  2. Selecciona "Compilar"
  3. Espera a que termine
  4. Luego reconstruye la solución completa
```

### Caso 2: "Error en PerfilDAL.cs"

```
CAUSA: Probablemente falta algún using
SOLUCIÓN:
  1. Abre Acceso_DAL/PerfilDAL.cs
  2. Verifica que tenga "using Entidad_BE;"
  3. Si no lo tiene, agrégalo arriba

Debe verse así:
????????????????????????????????????
? using System;                    ?
? using System.Collections.Generic;?
? using System.Data;               ?
? using Entidad_BE;  ? IMPORTANTE ?
? using Servicios;                 ?
?                                  ?
? namespace Acceso_DAL             ?
? {                                ?
?     public class PerfilDAL       ?
?     {                            ?
????????????????????????????????????
```

### Caso 3: "El proyecto está bloqueado"

```
CAUSA: Visual Studio tiene bloqueado el archivo
SOLUCIÓN:
  1. Cierra Visual Studio completamente
  2. Abre Administrador de Tareas
  3. Busca "devenv.exe" o "Visual Studio"
  4. Finaliza el proceso
  5. Vuelve a abrir Visual Studio
  6. Intenta compilar de nuevo
```
